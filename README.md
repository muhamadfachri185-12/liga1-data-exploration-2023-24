# liga1-data-exploration-2023-24
This dashboard visualizes key insights from the 2023/24 Liga 1 season, including player distribution by club, position, and nationality. It also highlights player age trends and market value comparisons, helping users understand league composition, talent diversity, and overall player value landscape.

### ðŸŽ¯ Goals
The goal of this project is to present a clear and interactive overview of player demographics and market valuations across all participating clubs in the Liga 1 2023/24 season.  
This dashboard aims to help users explore insights such as:

- Distribution of players by club, position, and nationality  
- Age profiles across clubs and roles  
- Comparison of player market values at both individual and team levels  

The visualizations are intended to make the data more accessible, insightful, and engaging for analysts, fans, or anyone interested in football data insights.
